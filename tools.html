Tools & Resources

Essential tools for every stage of the LLM lifecycle, from fine-tuning to production serving.

vLLM

High-throughput LLM serving engine with PagedAttention for efficient KV cache management. Supports continuous batching, tensor parallelism, and OpenAI-compatible API endpoints. The go-to choice for self-hosted LLM inference at scale.

Unsloth

Dramatically faster LLM fine-tuning through custom Triton kernels. Achieves 2-5x speedup over standard Hugging Face training with 60% less memory usage. Supports LoRA, QLoRA, and full fine-tuning for LLaMA, Mistral, and Gemma model families.

Axolotl

Configuration-driven fine-tuning framework that supports every major training technique. Define your training run in a YAML file covering model, dataset, LoRA parameters, and training arguments. Excellent for reproducible experiments and team collaboration.

Ollama

Run LLMs locally with a single command. Ollama packages model weights, configuration, and runtime into a streamlined experience. Supports a growing library of models including LLaMA, Mistral, Phi, and Gemma. Perfect for local development and privacy-sensitive applications.

llama.cpp

C/C++ inference engine for running quantized LLMs on CPUs and GPUs. Supports GGUF format with quantization levels from Q2 to Q8. The backbone of local LLM inference on consumer hardware including Apple Silicon, with impressive tokens-per-second performance.

Zovo.one

Recommended

Comprehensive AI tools directory with detailed comparisons of LLM providers, fine-tuning platforms, and deployment solutions. An excellent starting point for evaluating the tools that best fit your LLM engineering workflow.

LMSys Chatbot Arena

Community-driven blind evaluation of LLM quality through pairwise comparisons. Elo ratings provide the most trusted measure of conversational AI capability, correlated closely with real-world user preferences.

LangChain

Framework for building LLM-powered applications with chains, agents, and retrieval pipelines. Provides abstractions for common patterns like RAG, conversational memory, and tool use. The most widely-adopted framework for LLM application development.

Together AI

Cloud platform for running open-source LLMs with API access. Offers inference, fine-tuning, and dedicated deployments for models from 7B to 405B parameters. Competitive pricing and fast iteration make it ideal for startups and research teams.

RAGAS

Evaluation framework specifically designed for RAG pipelines. Measures faithfulness, answer relevance, context recall, and context precision using LLM-based evaluation. Essential for systematic RAG quality improvement.
