# AI Model Database — Browse 40+ AI Models with Specs, Benchmarks & Comparisons

**[Browse Models →](https://gpt0x.com)** | [About](https://gpt0x.com/about.html) | [Blog](https://gpt0x.com/blog/)

AI Model Database is a searchable directory of 40+ AI models from OpenAI, Anthropic, Google, Meta, Mistral, and more. Filter by model type (LLM, vision, embedding, multimodal), provider, parameter count, and license. Each model has an expandable card with architecture details, benchmark scores, pricing, and recommended use cases. The most comprehensive free AI model reference on the web.

## Features

- **40+ AI models indexed** — GPT-4, Claude, Gemini, Llama, Mistral, Stable Diffusion, and more
- **Filter by type, provider, and parameters** — narrow down models by what matters to your project
- **Expandable model cards** — architecture, training data, benchmarks, pricing, and use cases
- **Side-by-side comparison** — pin models and compare specs in a unified table
- **Regular updates** — new models added as they launch

## How It Works

Open the database and browse the full model catalog or use the filter panel to narrow by type (LLM, vision, embedding, multimodal), provider, parameter range, or license. Click any model to expand its full card with architecture details, key benchmark scores (MMLU, HumanEval, HellaSwag), API pricing, and recommended use cases. Pin multiple models to compare them side by side. All data loads client-side for instant filtering.

## Built With

- Vanilla JavaScript (no frameworks, no dependencies)
- Client-side only — your data never leaves your browser
