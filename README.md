# GPT0X

> Free AI model database and comparison tool — GPT-4, Claude, Gemini, Llama, Mistral, and 40+ LLMs side-by-side.

**[Use GPT0X live](https://gpt0x.com/)**

GPT0X is a searchable database of 40+ AI models from OpenAI, Anthropic, Google, Meta, Mistral, and more. Compare GPT-4, Claude, Gemini, Llama, and other models by parameters, context window, release date, and open-source status. Filter, sort, and find the right model for your project in seconds — no sign-up required.

## Features

- Database of 40+ AI models with detailed specifications
- Filter by modality: text, image, audio, multimodal
- Filter by provider: OpenAI, Anthropic, Google, Meta, Mistral, Cohere, and more
- Filter by open-source vs. closed-source status
- Sortable columns for every specification field
- Detailed model cards showing context window, parameters, pricing, and release date
- Quick search with instant results
- Cross-links to each provider's official documentation
- 100% client-side — no data leaves your browser
- MIT licensed
- No signup, no tracking

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript
- No build step
- No external dependencies (except Google Fonts on some pages)
- Hosted on GitHub Pages with Cloudflare CDN
