# GPT0X -- AI Model Database & Comparison Tool

**[-> Use GPT0X (live tool)](https://gpt0x.com/)**

GPT0X is a searchable database of 40+ AI models from OpenAI, Anthropic, Google, Meta, Mistral, and more. Compare GPT-4, Claude, Gemini, Llama, and other models by parameters, context window, release date, and open-source status. Filter, sort, and find the right model for your project instantly.

## Features

- Database of 40+ AI models with detailed specifications
- Filter by type: text, image, audio, multimodal
- Filter by provider: OpenAI, Anthropic, Google, Meta, Mistral, and more
- Filter by open-source vs. closed-source status
- Sortable columns for every specification field
- Detailed model cards with full specs accessible in one click
- 100% client-side -- your data never leaves your browser
- Open source -- inspect the code yourself

## Tech Stack

- Vanilla JavaScript (no frameworks, no build step)
- Static HTML hosted on GitHub Pages
- Cloudflare DNS + SSL
- Zero dependencies, zero tracking, zero cookies
